## âŒ¨ï¸ How to Use

TypeTalk AI uses a **Layered Command System** to avoid conflict with other NVDA shortcuts.

### Step 1: Activate Command Mode
Press the following key combination to enter the command layer:
> **NVDA + Shift + Space**

*(You will hear a sound indicating the layer is active)*

### Step 2: Press a Command Key
Once the layer is active, press one of the keys below:

| Key | Function | Description |
| :---: | :--- | :--- |
| **S** | **Start / Stop** | Toggle the microphone for voice typing. |
| **A** | **Toggle AI** | Switch between Smart AI Mode and Offline Mode. |
| **P** | **Punctuation** | Toggle Spoken Punctuation (Works in Offline Mode). |
| **T** | **Translation** | Enable/Disable translation to the target language. |
| **E** | **Emoji Mode** | Enable/Disable automatic emoji insertion. |
| **M** | **Change Model** | Switch AI models (GPT-5.1, Gemini Pro, Custom). |
| **I** | **Input Language** | Change your speaking language (e.g., Bengali, English). |
| **L** | **Target Language** | Change the language for translation. |
| **H** | **Help** | Open this documentation window. |
| **D** | **Developer** | Show information about the developer. |

You can customize TypeTalk AI settings from the NVDA menu:
