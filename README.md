## ğŸ® How to Use (Shortcuts)

TypeTalk AI uses a **Command Layer** system to prevent keyboard conflicts.

**Step 1:** Activate Command Mode:
> Press **`NVDA + Shift + Space`** (You will hear a start sound).

**Step 2:** Press one of the following keys:

| Key | Function | Description |
| :---: | --- | --- |
| **S** | **Start/Stop Dictation** | Starts listening. Press again to process text. |
| **A** | **Toggle AI Mode** | Turns AI grammar correction ON/OFF. |
| **P** | **Spoken Punctuation** | Turns manual punctuation ON (Only works when AI is OFF). |
| **T** | **Toggle Translation** | Enables/Disables translation mode. |
| **E** | **Toggle Emoji** | Adds relevant emojis to your text automatically. |
| **I** | **Input Language** | Opens a menu to select your speaking language. |
| **L** | **Target Language** | Select the language for translation. |
| **M** | **Change Model** | Switch between GPT, Gemini, or Custom API. |
| **H** | **Help Menu** | Shows the full list of commands. |
| **D** | **About** | Shows developer information. |
